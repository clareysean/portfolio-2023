Projects.

Jade Chat:

Jade Chat is a fully functional instant messenger for desktop, built using the MERN stack. The application uses web sockets via socket.io to enable real-time messaging between users. Jade Chat lets users add and remove contacts to their chat room and upload a custom avatar. The front-end of this application is built following optimistic-UI principles. The back-end follows an MVC architectural structure.

Stack:
 React js MongoDB AWS S3 Heroku Socket.io Node js Express js

My Health Console:

MyHealthConsole is a platform designed to help users keep track of their healthcare-related information. On the app users can track their care providers, prescriptions, or appointments. Our app provides a convenient and organized solution for all your healthcare data tracking needs. Built using the Django web framework and powered by PostgreSQL, this app ensures security, reliability, and a seamless user experience.

Stack:
 Django PostgreSQL AWS S3 Neon Python 3

Coffee DB:

CoffeeDB is a web app where users can add a post about their favorite coffee roasts to a database, review existing roasts, edit their posts and reviews, and build a favorites list of roasts. This project was inspired by the site Beer Advocate.

Stack:
 EJS MongoDB Heroku Mongoose Node js Express js
